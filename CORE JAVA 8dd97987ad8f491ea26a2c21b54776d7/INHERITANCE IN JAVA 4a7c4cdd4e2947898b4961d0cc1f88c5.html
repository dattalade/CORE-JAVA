INHERITANCE IN JAVA

INHERITANCE

  • Inheritance is a mechanism that acquires all the properties / functionalities from derived class to the base class
  • For understanding : derived class is known as parent class , base class is known as child class
  • It is known as 'IS-A' relationship
  • The DerivedClass is known as 'SubClass' or 'ChildClass'
  • The BaseClass is known as 'SuperClass' or 'ParentClass'
  • To implement Inheritance we use a keyword named extends
  • There are five types of Inheritances :
    1. Single Inheritance
    1. Multilevel Inheritance
    1. Hierarchial Inheritance
    1. Multiple Inheritance // Not supported in Java Classes
    1. Hybrid Inheritance

      Single Inheritance

      • Only derived class acquires properties from one base class.

      Diagram →

      flowchart LR
    subgraph BASE_CLASS
        CLASS_A
    end
    subgraph DERIVED_CLASS
        CLASS_B
    end
    CLASS_B --> CLASS_A

      Syntax →

      class BaseClass
{
	//code
}

class DerivedClass extends BaseClass
{
	//code
}

      Example_1 (Programmer.java) →

      package com.inheritance;
import java.util.*;

class Employee
{
	 float salary = 40000;
}

public class Programmer extends Employee
{
	int bonus = 10000;

	public static void main(String args[])
	{
		Programmer p = new Programmer();
		System.out.println("Employee salary is: "+p.salary);
		System.out.println("Bonus of Programmer is: "+p.bonus);
	}
}

      Output →

      Employee salary is 40000
      Bonus of Programmer is 10000

      MultiLevel Inheritance

      • One class acquires properties from one class that class acquires properties from another class and so on
      • That means, the last derived class acquires all properties of previous classes

      Diagram →

      flowchart LR
    subgraph BASE_CLASS
        CLASS_A
    end
    subgraph DERIVED_CLASS
        CLASS_B
				CLASS_C
				CLASS_D
		end
		
    CLASS_B --> CLASS_A
		CLASS_C --> CLASS_B
		CLASS_D --> CLASS_C

      Syntax →

      class BaseClass
{
	//code
}

class DerivedClass1 extends BaseClass
{
	//code
}

class DerivedClass2 extends DerivedClass1
{
	//code
}

-----------------

class DerivedClassn extends DerivedClassx
{
	//code
}

      package com.inheritance;
import java.util.*;

class Shape
{
  public void display()
	{
	  System.out.println("Shape");
  }
}

  public void area()
	{
		System.out.println("Square");
  }
}

{
	public void volume()
	{
	  System.out.println("Cube");
  }
}
public class Tester
{
	public static void main(String[] arguments)
	{
      Cube cube = new Cube();
      cube.display();

  }

      Output →

      Shape

      Square

      Cube
