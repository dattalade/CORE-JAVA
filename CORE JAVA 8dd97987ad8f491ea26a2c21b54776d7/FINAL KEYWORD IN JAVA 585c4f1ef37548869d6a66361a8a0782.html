FINAL KEYWORD IN JAVA

Final keyword

  • The final keyword in java is used to restrict the user
  • The java final keyword can be used in many context. Final can be:
    1. Variable
    1. Method
    1. Class

    Java Final variable

    • If you make any variable as final, you cannot change the value of that final variable
    • It will be a constant value

    Example_1 (Final.java) →

    package com.finalkeyword;
import java.util.*;

public class Final
{  
	final int speedLimit = 65;  
 
	void changeLimit()
	{
		speedLimit = 100;
	}

	public static void main(String args[])
	{
		Final f = new Final();
		f.changeLimit();
	}
}

    Output →

    error: cannot assign a value to final variable speedlimit

    Example_2 (Final.java) →

    package com.finalkeyword;
import java.util.*;

public class Final

    even static variables also default assigned with zero.

    Java Final method

    • If you make any method as final, you cannot override the method

    Example_1 (Final.java) →

    package com.finalkeyword;
import java.util.*;

class Original
{
	final void changeLimit()
	{
		System.out.println("Original Class");
	}
}
public class Final extends Original
{  
	void changeLimit()
	{
		System.out.println("Final Class");
	}

	public static void main(String args[])
	{
		Final f = new Final();
		f.changeLimit();
	}
}

    Output →

    error: changeLimit() in Final cannot override changeLimit() in Original

    void changeLimit(){System.out.println("Final class");}
        ^
  overridden method is final

    Java Final Class

    • If you make any class as final, you cannot inherit any base class / super class / parent class

    Example_1 (Final.java) →

    package com.finalkeyword;
import java.util.*;

final class Original
{
	void changeLimit()
	{
		System.out.println("Original Class");
	}
}
public class Final extends Original
{  
	void changeLimit()
	{
		System.out.println("Final Class");
	}

	public static void main(String args[])
	{
		Final f = new Final();
		f.changeLimit();
	}
}

    Output →

    error: cannot inherit from final Bike

    Note : Constructor cannot be declared as final because constructor is never inherited.
