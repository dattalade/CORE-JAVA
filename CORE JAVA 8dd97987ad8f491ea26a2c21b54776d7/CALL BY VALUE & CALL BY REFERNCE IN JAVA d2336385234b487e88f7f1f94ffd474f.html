CALL BY VALUE & CALL BY REFERNCE IN JAVA

Call by Value

  • In Java, call by value refers to calling a method by passing the value in the parameter.
  • In Java, call by value passes a copy of the variable to the method, so all changes are reflected only in that method, thus no changes done in caller’s method.
  • When we pass a variable with primitive data types, it is considered as call by values in java, so any changes to the variable will not be reflected in the caller’s method

Example_1 (CallByValue.java) →

package com.callbyvalue;
import java.util.*;

public class CallByValue
{
    public static void increment(int number)
		{
        number = number+1;
        System.out.println("Value in method : "+number);
    }
    
    public static void main(String[] args) {
        int number=10;
        System.out.println("Value before method call : "+number);
        increment(number);
        System.out.println("Value after method call : "+number);
    }
}

Output →

Value before method call : 10

value in method : 11

Value after method call : 10

Call by Reference

  • In Java, call by reference refers when we pass the reference of the object, it creates a copy of the reference and then passes it as value to the method.
  • The copy reference also points to the same address so all the changes also reflect in the caller’s method

Example_1 (CallByReference.java) →

package com.callbyreference;
import java.util.*;

public class CallByReference
{
    int number = 10;

    public static void increment(CallByReference cr)
		{
        cr.number = cr.number+1;  // increment variable by 1
        System.out.println("Value in method: " + cr.number);

Example_2 (CallByReference.java) →

package com.callbyreference;
import java.util.*;

public class CallByReference
{
    int number = 10;

    public static void increment(int arr[])
		{
        arr[0] = arr[0]+1;  // increment variable by 1
        System.out.println("Value in method: " + arr[0]);
    }
    
    public static void main(String[] args)
		{
				int arr[] = new int[1];
				arr[0] = 10;
        System.out.println("Value before method call: " + arr[0]);
        increment(arr);
        System.out.println("Value after method call: " + arr[0]);
    }
}

Output →

Value before method call : 10

value in method : 11

Value after method call : 11
