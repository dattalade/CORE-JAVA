CONTROL STATEMENTS IN JAVA

Control Statements

  • Java runs a code line by line, if we use control statements we can execute a code without following a rule 'line by line'
  • In simpler words, Java uses Control Ststements to cause or disturb the flow of Execution
  • There are three types of Control Statements
    1. Decision-Making Statements
    1. Iterative Statements
    1. Jump Statements

      Decision Making Statements

      Types of Decision Making Statements are :

      1. Simple-if Statement
      1. if-else Statement
      1. else-if ladder
      1. Nested if Statement
      1. Switch Statement

        Simple-if

        • If the given condition / conditions are true, then program will execute that 'if' block of code.

        Syntax → 

        if(condition (or) conditions)
{
	//code
}

        Example_1 (SimpleIf.java) →

        package com.decision;
import java.util.*;

public class SimpleIf
{
	public static void main(String[] args)
	{
		int a = 10, b =20;
		if(a < b)
		{
				System.out.println("Big Number");
		}
	}
}

        Output →

        Big Number

        if-else

        • If the given condition / conditions are true, then program will execute 'if' block of code.
        • If the given condition fails, then program will execute 'else' block of code.

        Syntax → 

        if(condition (or) conditions)
{
	//code
}
else
{
	//code
}

        Example_1 (IfElse.java) →

        package com.decision;
import java.util.*;

public class IfElse
{
	public static void main(String[] args)
	{
		int a = 10, b =20;
		if(a < b)
		{
			System.out.println("Big Number");
		}
		else
		{
			System.out.println("Small Number");
		}
	}
}

        Output →

        Big Number

        Example_2 (IfElse.java) →

        else-if Ladder

        • If the given condition / conditions are passed in 'if' block, then program will execute that 'if' block of code
        • If the condition is failed , it goes to next 'else if' code
        • If any of the block condition passes then it stops executing other blocks (till else block)

        Syntax → 

        if(condition (or) conditions)
{
	//code
}
else if(condition / conditions)
{
	//code
}
else if(condition / conditions)
{
	//code
}
else if(condition / conditions)
{
	//code
}
....................
else
{
	//code
}

        Example_1 (Ladder.java) →

        package com.decision;
import java.util.*;

public class Ladder
{
	public static void main(String[] args)
	{
		int a = 10, b =20;
		if(a > b)
		{
			System.out.println("Big Number");
		}
		else if(b%a == 0 && a%b == 10)
		{
			System.out.println("Remainder Number");
		}
		else
		{
			System.out.println("No Number");
		}
	}
}

        Output →

        Remainder Number

        Nested If

        • If the given condition / conditions are true, then program will execute 'if' block of code.
        • Then it checks the condition inside the 'if' block , going on till last 'if' block of code.

        Syntax →

        if(condition (or) conditions)
{
	if(condition (or) conditions)
	{
		if(condition (or) conditions)
		{
			..................
			//code
		}
	}
}

        Example_1 (Nested.java) →

        package com.decision;
import java.util.*;

public class IfElse
{
	public static void main(String[] args)
	{
		int a = 10, b =20;
		if(a < b)
		{
			System.out.println("First if passed");
			if(a-b == -10)
			{
				System.out.println("Second if passed");
				if(b%a == 1)
				{
					System.out.println("All if's passed");
				}
			}
		}
	}
}

        Output →

        Second if passed

        Switch Statements

        Syntax →

        switch(value)
{
	case val_a: // code
							break;
	case val_b: // code
					    break;
	case val_c: // code
					    break;
						------
	default : //code
}

        Example_1 (Switch.java) →

        package com.decision;
import java.util.*;

public class Switch
{
	public static void main(String[] args)
	{
		int a = 3;
		switch(a)
		{
			case 1: System.out.println("Sunday");
							break;
			case 2: System.out.println("Monday");
							break;
			case 3: System.out.println("Tuesday");
							break;
			case 4: System.out.println("Wednesday");
							break;
			case 5: System.out.println("Thursday");
							break;
			case 6: System.out.println("Friday");
							break;
			case 7: System.out.println("Saturday");
							break;
			default : System.out.println("No Day");						
		}
	}
}

        Output →

        Tuesday

        Example_2 (Switch.java) →

      Iterative Statements

      • To iterate a block of code multiple times or repeatedly, we use loops called Iterative Statements.
      • The block of iteration code iterates continously based on given condition when it meets given requirements.
      • We can also code Decision making statements inside Iterative Statements.
      • There are four types of loops :
        1. for-loop
        1. while-loop
        1. do-while-loop
        1. for-each-loop (We will discuss this topic in Arrays)

          for-loop

          • If the given condition / conditions are true, then program will execute that 'if' block of code.

          Syntax → 

          for(initialization;condition (or) conditions;updation)
{
	//code
}

          • Initialization : We have to declare a variable (with some datatype) with value
          • Condition : If the given condition fails, the loop stops iterating
          • Updation : We update a variable (either initialization variable or any other variable)

          Example_1 (ForLoop.java) →

          package com.iterative;
import java.util.*;

public class ForLoop
{
	public static void main(String[] args)
	{
		for(int i=1;i<=5;i++)
		{
			System.out.print(i + " ");
		}
	}
}

          Output →

          1 2 3 4 5

          Example_2 (ForLoop.java) →

          package com.iterative;
import java.util.*;

public class ForLoop
{
	public static void main(String[] args)
	{
		for(int i=1;i<=10;i++)
		{
			if(i%2 == 0)
			{
				System.out.print(i + " ");
			}
		}
	}
}

          Output →

          2 4 6 8 10

          While Loop

          • The disadvantage of for-loop is we have to know how many iterations we need
          • When iterations are not fixed, it is recommended to use while loop.

          Syntax → 

          initialization;
while(condition)
{
	// code
	updation;
}

          Example_1 (While.java) →

          package com.iterative;
import java.util.*;

public class IfElse
{
	public static void main(String[] args)
	{
		int n = 5;
		while(n>=0)
		{
			if(n%2 == 1)
				n--;
			else
				n = n/2;
			System.out.print(n + " ");
		}
	}
}

          Output →

          4 2 1 0

          Do While Loop

          • When iterations are not fixed but the loop must execute atleast once, it is recommended to use while loop.

          Syntax → 

          initialization;

do
{
	//code
	updation;
}while(condition (or) conditions);

          Example_1 (DoWhile.java) →

          package com.iterative;
import java.util.*;

public class DoWhile
{
	public static void main(String[] args)
	{
		int n = 1;
		do
		{
			if(n%2 == 1)
				n = n/2;
			else
				n--;
			System.out.println(n + " ");
		}while(n!=1);
	}
}

          Output →

          0

      Jump Statements

      • To iterate a block of code repeatedly we use loops, but to disturb the flow of loop we use Jump Statements
      • There are three types of loops :
        1. break
        1. continue
        1. return (will discuss in methods concept)

          break

          • If you want to stop upcoming iterations of the loop based on a condition , we use break keyword.

          Example_1 (Break.java) →

          package com.jumps;
import java.util.*;

public class Break
{
	public static void main(String[] args)
	{
		for(int i=1;i<=5;i++)
		{
			if(i%4 == 0)
			{
				break;
			}
			Sysem.out.print(i + " ");
		}
	}
}

          Output →

          1 2 3

          continue

          • If you want to skip the current iteration based on condition, we use continue keyword.Syntax →

          Example_1 (Continue.java) →

          package com.jumps;
import java.util.*;

public class Continue
{
	public static void main(String[] args)
	{
		for(int i=1;i<=5;i++)
		{
			if(i == 3)
			{
				continue;
			}
			Sysem.out.print(i + " ");
		}
	}
}

          Output →

          1 2 4 5
