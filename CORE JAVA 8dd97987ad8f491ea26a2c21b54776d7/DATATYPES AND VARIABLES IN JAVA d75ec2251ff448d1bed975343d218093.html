  1. int/short/byte/long : It stores only integer values (5, 10, 54, 23, etc....)
  1. double/float : It stores only decimal point values (2.33, 7.64, 9.89, 32.558, etc...)
  1. char : It stores only single character ('A', 'B', 'd', '/', '>', '+', '$', '1', etc...)
    Note - It must be declared inside single quotes only.
  1. String : It stores group of characters or single characters ("Apple", "123@123as", "A", etc...)
    Note - It must be declared inside double quotes only.
  1. boolean : It stores either 'true' or 'false'.
Note : All the datatypes above are primitive datatypes except String
Note : String is non-primitive datatype, it is a Class

Variables

  • Variables are used to store any value of any type for future purpose or use.
  • We can declare a variable by giving a name to it.
  • We can variables as Identifiers because it has it’s own identifiaction.
  • Variables are called as Data Members or Data Attributes
  • There are three types of variables in Java:
    1. Local variable
    1. Instance variable
    1. Static variable
Note : For now, we will see only about local Variables

Syntax → dataType variable_Name; (or) dataType variable_Name = value;

Example → int a; (or) int a = 10;

Local variable

  • A Variable declared inside the body of the method is called local variable
  • You can use this variable only within that method and the other methods in the class aren't even aware that the variable exists.

Restrictions

  • A variable name must not be a keyword.
  • A variable name must start with alphabets (either uppercase or lowercase) or with an underscore symbol '_'.
  • A variable name must not start with any other characters (symbols or numbers).

Declaring Multiple variables of same DataType

Syntax → dataType variable_Name1, variable_Name2 (or) dataType variable_Name1 = value1, variable_Name2 = value2;

Example → int a, b; (or) int a = 10, b = 20;

Example_1 (Variables.java) → 

package com.variables;
import java.util.*;

public class Variables
{
	public static void main(String[] args)
	{
		int a, b;
		a = 10;
		b = 20;
		System.out.println(a + " " + b);
	}
}

Output →

10 20

Example_2 (Variables.java) → 

package com.variables;
import java.util.*;

public class Variables
{
	public static void main(String[] args)
	{
		int a;
		System.out.println(a);
	}
}

Output →

error: variable a might not have been initialized

Example_3 (Variables.java) → 

package com.variables;
import java.util.*;

public class Variables
{

	public static void main(String[] args)
	{
		int enum = 10;
		System.out.println(enum);
	}
}

Output →

error: 'enum' is a keyword, and may not be used as an identifier

Example_4 (Variables.java) → 

package com.variables;
import java.util.*;

public class Variables
{
	public static void main(String[] args)
	{
		int var = 10;
		var = 20;
		System.out.println(var);
	}
}

Output →

20

Note : The variables which we declared in static methods called as local variables and might not have initialized.
