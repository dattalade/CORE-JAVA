INTRODUCTION TO JAVA

What is Java?

  • Java is a Programming language which is secured and object-oriented.
  • To run a java program we need JDK (Java Development Kit)
  • Under JDK we have two packages names JRE and JVM
  • It is platform independent

History of Java

  • Java is developed by Sun Micro Systems in the year 1995
  • James Gosling is the one who invented java (Father of Java)
  • The first version of java is 'jdk 1.0' in January 1996.
  • The current version of java is 'jdk 20' in March 2023.
  • We use 'jdk 17'
Note : Java files must be saved in '.java' extension

Feautres of Java

  • Platform Independence → Java's bytecode can be executed on any platform with an appropriate JVM
  • Object Oriented → Java also follows object-oriented programming
  • Simple & familiar → Java syntax is familiar to C & C++ which makes comfortable to programmers
  • Robust → Java programs ensures they can handle unexpected errors without causing harm
  • Secure → Java secures the data using access modifiers
  • MultiThreading → Java allows multiple tasks to run concurrently
  • Dynamic & high performance
  • Portable → Java's 'write once, run anywhere' capability makes it highly portable

Applications of Java

  • Web Applications
  • Mobile Applications
  • Desktop Applications

JDK (Java Development Kit)

  • It is a software development environment which is used to develop Java applications.
  • JDK is the implementation of below platforms :
    1. Standard Edition Java Platform
    1. Enterprise Edition Java Platform
    1. Micro-Edition Java Platform
  • JDK contains:
    1. Java Runtime Environment (JRE)
    1. Interpreter / Loader (Java)
    1. Compiler (javac)

JRE (Java Runtime Environment)

  • JRE is a software environment that is required for executing Java applications on any system regardless of operating system.
  • It cannot be used for developing, it is only used for executing those developed applications.

Keywords in Java

  1. Data Types : byte, short, int, long, float, double, char, boolean
  1. Access Modifiers : private, protected, public, default
  1. Control Statement : if, else, switch, case, default, for, while, do, continue, break, goto
  1. Object and Classes : class, interface, extends, implements, this, super, new
  1. Modifiers : static, abstract, synchronized, volatile, native, transient, strictfp, final
  1. Exceptions : try, catch, finally, throw, throws

Comments in Java

  • There are two types of comments in Java :
    1. Single-Line Comment
    1. Multi-Line Comment

Example →

public class Main
{
	public static void main(String[] args)
	{
		// This is a Single Line Comment

		/*
			This is Multi Line Comment
			We can write a paragraph here
		*/
	}
}
  • The line which is written after '//' is not considered as code of a program
  • The lines which are written in between '/*' and '*/' are not considered as code of a program
