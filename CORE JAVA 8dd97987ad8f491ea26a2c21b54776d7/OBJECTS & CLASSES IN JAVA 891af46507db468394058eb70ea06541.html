OBJECTS & CLASSES IN JAVA

Objects in Java

  • The Entity which have a State, Behaviour and an Identity is called as Object.
  • State has a value which is called as Data Member.
  • Behaviour has functionality called as Methods (Object Methods).
  • Every class can have multiple objects, for every object, state is different (Identity).
  • For Example, let us assume an Debit Card as our Object.
    • The Debit Card Number / CVV is called as State
    • WithDrawing the money or Depositing the money is called Behaviour
    • It has it's own Identity.
  • There are many ways to create an object in java. They are:
    1. By new keyword
    1. By newInstance() method
    1. By clone() method
    1. By deserialization
    1. By factory method etc.
Note : 2, 3, 4, 5 will learn later

Syntax →

className object_Name = new className(/* parameterlist(optional) */);
  • new ClassName is creating a Anonymous Object.
  • Anonymous Object is stored in a variable className object_Name for future purpose.

Example (DebitCard.java) →

package com.objects;
import java.util.*;

public class DebitCard
{
	public static void main(String[] args)
	{
		DebitCard p1 = new DebitCard();
	}
}

Classes in Java

  • A class is a group of objects which have common properties.
  • It is a template or blueprint from which objects are created.
  • It is a logical entity. It can't be physical.
  • A Class in Java can contain :
    1. Fields (Data Attributes)
    1. Methods
    1. Constructors
    1. Blocks
    1. Nested Class & Interfaces

Syntax →

class class_name
{  
    fields;  
    methods;
		----------  
}

Instance variable in Java

  • A variable which is created inside the class but outside the method is known as an instance variable.
  • Instance variable doesn't get memory at compile time. It gets memory at runtime when an object or instance is created.

new keyword in Java

  • The new keyword is used to allocate memory at runtime. All objects get memory in Heap memory area.
Note : To access any object method or datamember we must use object reference variable

Example (ObjectAndClass.java) →

package com.objectclass;
import java.util.*;

public class Student
{  
	int id;
	String name;  
	public static void main(String args[])
	{  
		Student s1 = new Student();
	}  
}

Example_1 (SingleObject.java) →

package com.objectclass;
import java.util.*;

public class SingleObject
{  
	int id;
	String name;  
	public static void main(String args[])
	{  
		Student s1 = new Student();
		s1.id = 20;
		s1.name = "datta";
		System.out.println(s1.id + " " + s1.name);
	}  
}

Output →

20 datta

Example_2 (MultipleObjects.java) →

package com.objectclass;
import java.util.*;

public class MultipleObjects
{  
	int id;
	String name;  
	public static void main(String args[])
	{  
		Student s1 = new Student();
		Student s2 = new Student();

		s1.id = 20;
		s1.name = "datta";

		s2.id = 40;
		s2.name = "ntr";

		System.out.println(s1.id + " " + s1.name);
		System.out.println(s2.id + " " + s2.name);
	}  
}

Output →

20 datta

40 ntr

  • For every Object, we have different values for each Data Member
