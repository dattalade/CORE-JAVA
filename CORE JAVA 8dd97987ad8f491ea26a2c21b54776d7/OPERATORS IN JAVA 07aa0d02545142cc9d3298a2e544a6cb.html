OPERATORS IN JAVA

Operators in Java

There are eight operators :

  1. Unary Operators
  1. Arithmetic Operators
  1. Bitwise Operators
  1. Relational Operators
  1. Logical Operators
  1. Ternary Operators
  1. Assignment Operators

    Unary Operators

    • To perform any operations in unary, we need only one operand
    • Types of Unary Operators are →
      1. Increment
      1. Decrement
      1. Not

      Increment (++)

      • The value of a variable will be increased by 1
      • There are two types of Increment :
        1. Post-Increment (First it will assign and then it will be incremented)
        1. Pre-Increment (First it will increments and then assigns)

        Example_1 (PostIncrement.java) →

        package com.operators;
import java.util.*;

public class PostIncrement
{
	public static void main(String[] args)
	{
		int x = 10;
		x++;	// Post Increment
		System.out.print(x + " ");		// x = 11
		System.out.print(x++ + " ");	// x = 11
		System.out.println(x + " ");		// x = 12

		int y = x++;
		System.out.print(y + " ");		// y = 12
		System.out.println(x);		// x = 13
	}
}

        Output →

        11 11 12

        12 13

        Example_2 (PreIncrement.java) →

        package com.operators;
import java.util.*;

public class PreIncrement
{
	public static void main(String[] args)
	{
		int x = 10;
		++x;	// Pre Increment
		System.out.println(x)		// x = 11
		System.out.println(++x)		// x = 12
		System.out.println(x)		// x = 12
	}
}

        Output →

        11 12 12

      Decrement (—)

      • The value of a variable will be decreased by 1
      • There are two types of Decrement :
        1. Post-Decrement (First it will assign and then it will be Decremented)
        1. Pre-Decrement (First it will Decrements and then assigns)

        Example_1 (PostDecrement.java) →

        package com.operators;
import java.util.*;

public class PostDecrement
{
	public static void main(String[] args)
	{
		int x = 10;
		x--;	// Post Decrement
		System.out.println(x)		// x = 9
		System.out.println(x--)	// x = 9
		System.out.println(x)		// x = 8

		int y = x--;
		System.out.println(y)		// y = 8
		System.out.println(x)		// x = 7	
	}
}

        Output →

        9 9 8

        8 7

        Example_2 (PreDecrement.java) →

        package com.operators;
import java.util.*;

public class PreDecrement
{
	public static void main(String[] args)
	{
		int x = 10;
		--x;	// Pre Decrement
		System.out.println(x)		// x = 9
		System.out.println(--x)	// x = 8
		System.out.println(x)		// x = 8
	}
}

        Output →

        9 8 8

      Not (!)

      • It will give an alternative answer

      Example_1 (Not.java) →

      package com.operators;
import java.util.*;

public class Not
{
	public static void main(String[] args)
	{
		boolean x = true;
		x = !x	// x = false
		System.out.println(x);
	}
}

      Output →

      false

      Example_2 (PreDecrement.java) →

      package com.operators;
import java.util.*;

public class PreDecrement
{
	public static void main(String[] args)
	{
		int x = 10;
		--x;	// Pre Decrement
		System.out.println(x)		// x = 9
		System.out.println(--x)	// x = 8
		System.out.println(x)		// x = 8
	}
}

      Output →

      9 8 8

    Arithmetic Operators

    • To perform any operations in arithmetic, we need atleast two operands
    • It is used to perform normal mathematical operations
    • Types of Arithmetic Operators are →
      1. Addition (+)
      1. Subtraction (-)
      1. Multiplication (*)
      1. Division (/)
      1. Modulus or Remainder (%)

      Example_1 (Arithmetic.java) →

      package com.operators;
import java.util.*;

public class Arithmetic
{
	public static void main(String[] args)
	{
		int a = 21;
		int b = 10;

		System.out.println("Addition of " + a + " and " + b + " is " + a+b);		// 31
		System.out.println("Subtraction of " + a + " and " + b + " is " + a-b);		// 11
		System.out.println("Multiplication of " + a + " and " + b + " is " + a*b);		// 210
		System.out.println("Division of " + a + " and " + b + " is " + a/b);		// 2
		System.out.println("Remainder of " + a + " and " + b + " is " + a%b);		// 1
	}
}

      Output →

      Addition of 21 and 10 is 31

      Subtraction of 21 and 10 is 11

      Multiplication of 21 and 10 is 210

      Division of 21 and 10 is 2

      Remainder of 21 and 10 is 1

    Bitwise Operators

    • It is used to manipulate the bits
    • Types of Bitwise Operators are →
      1. Shifting
      1. Logical

      Shifting

      • There are two types of Shifting :
        1. Left Shift (<<) → a << b → a * (2 power b)
        1. Right Shift (>>) → a >> b → a / (2 power b)

        Example_1 (Shifting.java) →

        package com.operators;
import java.util.*;

public class Shifting
{
	public static void main(String[] args)
	{
		System.out.println("Left Shift of 10 and 3" + " is" + 10 << 3);	// 80	(10 * (8))
		System.out.println("Right Shift of 10 and 2" + " is" + 10 >> 2);	// 2	(10 / (4))
	}
}

        Output →

        Left Shift of 10 and 3 is 80

        Left Shift of 10 and 2 is 2

      Logical

      • There are five types of Logical operators :
        1. and (&)
        1. or (|)
        1. xor (^)
        1. not (~)

        Example_1 (LogicalOperator.java) →

        package com.operators;
import java.util.*;

public class LogicalOperator
{
	public static void main(String[] args)
	{
		int a = 42;
		int b = 15;
		System.out.println(a & b)		// 10	(00101010 & 00001111)
		System.out.println(a | b)		// 47	(00101010 | 00001111)
		System.out.println(a ^ b)		// 37	(00101010 ^ 00001111)
	}
}

        Output →

        10

        47

        37

    Relational Operators

    • It checks the given single condition is true or not
    • Types of Relational Operators are →
      1. Greater than (>)
      1. Less than (<)
      1. Greater than or equals to (>=)
      1. Less than or equals to (<=)
      1. Instance (instanceof) // Will learn later in OOP's Concepts
      1. Equals (==)
      1. Not equals (!=)

      Example_1 (RelationalOperator.java) →

      package com.operators;
import java.util.*;

public class RelationalOperator
{
	public static void main(String[] args)
	{
		int x = 5;
    int y = 3;
    System.out.print(x > y + " ");
		System.out.print(x < y + " ");
		y = 5;
		System.out.print(x >= y + " ");
		System.out.print(x <= y + " ");
		System.out.print(x == y + " ");
		System.out.print(x != y);
	}
}

      Output →

      true false true true true false

    Logical Operators

    • It checks the given multiple condition are true or not
    • Types of Logical Operators are →
      1. Logical And (&&) : It returns true when all the given conditions are true, returns false when one of the given condition fails
      1. Logical Or (||) : It returns true if one in all given conditions becomes true, returns false when every condition fails

      Example_1 (LogicalOperator.java) →

      package com.operators;
import java.util.*;

public class LogicalOperator
{
	public static void main(String[] args)
	{
		int a=10;  
		int b=5;  
		int c=20;  

		System.out.print(a<b && a<c + " ");	//false && true = false
		System.out.println(a>b && b<c);	// true && true = true

		System.out.print(a<b || a<c + " ");	//false || true = true
		System.out.println(a==b || b>c);	// false || false = false
	}
}

      Output →

      false true

    Ternary Operator

    • The variable stores a value, based on the condition given in ternary operator

    Syntax → dataType var_name = condition?value_x:value_y;

    Example_1 (Ternary.java) →

    package com.operators;
import java.util.*;

public class Ternary
{
	public static void main(String[] args)
	{
		char a = 5>3?'y':'n';
		System.out.println(a);	// false || false = false
	}
}

    Output →

    y

    Assignment Operators

    • To use this operator the variable must be defined before and must have a previous value
    • It will operate value with existing value of the variable
    • These operators are also called as 'Compound Assignments’
    • Types of Assignment Operators are →
      1. Addition Assignment (+=)
      1. Subtraction Assignment (-=)
      1. Multiplication Assignment (*=)
      1. Division Assignment (/=)
      1. Modulus Assignment (%=)
      1. Equals Assignment (=)
      Note : These are only few operators , these can be used for bitwise operators too

    Syntax → dataType var_name = value_x;

    var_name (Assignment Operator) value_y;

    Example_1 (Assignment.java) →

    package com.operators;
import java.util.*;

public class Assignment
{
	public static void main(String[] args)
	{
		int a = 10;

		a += 20;
		System.out.print(a + " ")		// a = 30 (a += 20 -> a = a+20 -> a = 10+20 -> a = 30)

		a -= 20;
		System.out.print(a + " ")		// a = 10 (a -= 20 -> a = a-20 -> a = 30-20 -> a = 10)

		a *= 20;
		System.out.print(a + " ")		// a = 200 (a *= 20 -> a = a*20 -> a = 10*20 -> a = 200)

		a /= 10;
		System.out.print(a + " ")		// a = 20 (a /= 10 -> a = a/10 -> a = 200/10 -> a = 20)

		a %= 2;
		System.out.println(a)		// a = 0 (a %= 2 -> a = a%2 -> a = 20%2 -> a = 0)

		int p, q, r;
		p = q = r = 100;
		System.out.println(p + " " + q+r)
	}
}

    Output →

    30 10 200 20 0

    100 200
