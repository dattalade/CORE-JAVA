ARRAYS IN JAVA

Arrays in Java

  • Till now, we used to store only one value in a single variable, but to store multiple values (of same datatype) in one variable we use Arrays.
  • Every element is stored in single variable with it's sequential indexes (0 to length-1), i.e If the array size is n, Index lasts at n-1
  • Array is an derived datatype
  • Memory will allocate continously
  • In java, array is treated as Object. So, memory is allocated in heap.

Advantages of Arrays

  • Optimization of Code
  • We can get data(value) located at an index position

Dis-Advantages of Arrays

  • We can't resize the array, the size of an array variable is fixed

There are two types of arrays :

  1. Single Dimensional Arrays
  1. Multi Dimensional Arrays

    Single Dimensional Array

    Syntax →

    dataType[] array_Variable = new dataType[size];
										(or)
dataType array_Variable[] = new dataType[size];

    Example_1 (Arrays.java) →

    package com.arrays;
import java.util.*;

public class Arrays
{
	public static void main(String[] args)
	{
		int arr[] = new int[4];
		arr[0] = 1;
		arr[1] = 2;
		arr[2] = 3;
		arr[3] = 4;
		System.out.print(arr[0] + " " + arr[1] + " " + arr[2] + " " + arr[3]);
	}
}

    Output →

    1 2 3 4

    Example_2 (Arrays.java) →

    package com.arrays;
import java.util.*;

public class Arrays
{
	public static void main(String[] args)
	{
		int arr[] = new int[4];
		for(int i=0;i<4;i++)
		{
			arr[i] = i+1;
		}
		for(int i=0;i<4;i++)
		{
			System.out.print(arr[i] + " ");
		}
	}
}

    Output →

    1 2 3 4

    Multi-Dimensional Array

    • It is also known as matrix

    Syntax →

    dataType[][] array_Variable = new dataType[row_size][column_size];
										(or)
dataType array_Variable[][] = new dataType[row_size][column_size];

    Example_1 (Arrays.java) →

    package com.arrays;
import java.util.*;

public class Arrays
{
	public static void main(String[] args)
	{
		int arr[][] = new int[3][3];
		arr[0][0] = 10;
		arr[0][1] = 20;
		arr[0][2] = 30;
		arr[1][0] = 40;
		arr[1][1] = 50;
		arr[1][2] = 60;
		arr[2][0] = 70;
		arr[2][1] = 80;
		arr[2][2] = 90;
		System.out.println(arr[0][0] + " " + arr[0][1] + " " + arr[0][2]);
		System.out.println(arr[1][0] + " " + arr[1][1] + " " + arr[1][2]);
		System.out.println(arr[2][0] + " " + arr[2][1] + " " + arr[2][2]);
	}
}

    Output →

    10 20 30

    40 50 60

    70 80 90

    Example_2 (Arrays.java) →

    package com.arrays;
import java.util.*;

public class Arrays
{
	public static void main(String[] args)
	{
		int arr[][] = new int[3][3];
		for(int i=0;i<3;i++)
		{
			for(int j=0;j<3;j++)
			{
				arr[i][j] = i+j;
			}
		}
		for(int i=0;i<3;i++)
		{
			for(int j=0;j<3;j++)
			{
				System.out.print(arr[i][j] + " ");
			}
			System.out.println();
		}
	}
}

    Output →

    0 1 2

    1 2 3

    2 3 4

    Example_3 (Arrays.java) →

    package com.arrays;
import java.util.*;

public class Arrays
{
	public static void main(String[] args)
	{
		int arr[][] =new int[][]{{10, 20, 23}, {11, 12, 13}};

		System.out.println(arr[1][0] + " " + arr[1][1] + " " + arr[1][2]);
	}
}

    Output →

    10 20 23

    11 12 13

    Note : To find row_size → arrayName.length
    Note : To find column_size → arrayName[index].length

For-Each Loop in Java

Syntax →

for(datatype var_Name : array)
{
	//code
}

Example (ForEach.java) →

package com.foreach;
import java.util.*;

public class ForEach
{
	public static void main(String[] args)
	{
		int arr[] =new int[4];
		for(int i=0;i<arr.length;i++)
		{
			arr[i] = i+1;
		}
		for(int i:arr)
		{
			System.out.print(i + " ");
		}
	}
}

Output →

1 2 3 4
