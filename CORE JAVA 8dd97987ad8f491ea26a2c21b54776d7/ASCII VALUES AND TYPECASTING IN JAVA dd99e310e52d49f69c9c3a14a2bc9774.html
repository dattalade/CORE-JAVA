ASCII VALUES AND TYPECASTING IN JAVA

ASCII (Integer) values of every character

  1. [A-Z] : [65-90]
  1. [a-z] : [97-122]
  1. [0-9] : [48-57]
  1. [{|}~] : [123, 124, 125, 126]
  1. [[\]^] : [91, 92, 93, 94]

Example_1 (Ascii.java) → 

package com.ascii;
import java.util.*;

public class Ascii
{
	public static void main(String[] args)
	{
		char c = 'D'
		System.out.println((int)c);
	}
}

Output →

68

Type Casting in Java

There are two types of type casting :

  1. Widening Type Casting
  1. Narrow Type Casting

    Widening Type Casting

    • It converts from lower datatype to higher datatype.
    • It converts automatically while assigning.

    Example_1 (WTC.java) →

    package com.typecasting;
import java.util.*;

public class WTC
{
	public static void main(String[] args)
	{
		int a = 10;
		double b = a;		// b = 10.0
		System.out.println("Double value of " + a + " is" + b);
		char c = 'C';
		int d = c;		// d = 67
  	System.out.println("Integer value of " + c + " is" + d);
	}
}

    Output →

    Double value of 10 is 10.0

    Integer value of C is 67

    Example_2 (WTC.java) →

    package com.typecasting;
import java.util.*;

public class WTC
{
	public static void main(String[] args)
	{
		int a = 10;
		byte b = a;		// Cannot convert int to byte (lossy conversion)
		
		double c = 30.33;
		int d = c;			// Cannot convert double to int (lossy conversion)
	}
}

    Output →

    error: incompatible types: possible lossy conversion from int to byte

    error: incompatible types: possible lossy conversion from double to int

    Narrow Type Casting

    • It converts from lower datatype to higher datatype (or) viceversa.
    • We should manually convert into another datatype

    Example_1 (NTC.java) →

    package com.typecasting;
import java.util.*;

public class NTC
{
	public static void main(String[] args)
	{
		double a = 10.98;
		int b = (int) a;		// b = 10
		System.out.println("Integer value of " + a + " is" + b);
		int ch = 67;
    char i = (char)ch;		// i = C
		System.out.println("Character of this " + ch + " is" + i);
	}
}

    Output →

    Integer value of 10.98 is 10

    Character of this 67 is C
