STRING CLASS IN JAVA

String Class

  • String is a class where we can create an object to store sequence of characters
  • String are immutable. i.e, where strings cannot be manipulated
  • String is available in java.lang package
  • String class extends Object and implements Serializable, CharSequence, Comparable<String> and String is final.
    public final class String extends Object implements Serializable, CharSequence, Comparable<String>

String Constructors

ConstructorDescription
String()Initializes a newly created String object so that it represents an empty character sequence.
String(byte[] arr)Constructs a new String by decoding the byte array
String(byte[] arr, int start_index, int length)Constructs a new String for a given byte array, from specified index to index+length
String[char[] arr)Constructs a new String for given sequence of characters
String(char[] arr, int start_index, int length)Constructs a new String for a given char array, from specified index to index+length
String(int[] arr, int start_index, int length)Constructs a new String for a given integer array if and only if we specify start_index and length
String(StringBuilder sb)Allocates a new String
String(StringBuffer sb)Allocates a new String
Note : The default encoding charset is UTF-8

Example_1 (StringConstructors.java) →

package com.strings;
import java.util.*;

public class StringConstructors
{  
	public static void main(String args[])
	{  
	  String s1 = new String();
		System.out.println("First Constructor Response : " +s1);

		byte[] arr = new byte[]{65, 66, 67, 68};
		String s2 = new String(arr);
		System.out.println("Second Constructor Response : " +s2);

		byte[] arr1 = new byte[]{65, 66, 67, 68};
		String s3 = new String(arr1, 1, 3);
		System.out.println("Third Constructor Response : " +s3);

		char[] ch = new char[]{'d', 'a', 't', 't', 'a'};
		String s4 = new String(ch);
		System.out.println("Fourth Constructor Response : " +s4);

		char[] ch1 = new char[]{'d', 'a', 't', 't', 'a'};
		String s5 = new String(ch1, 1, 3);
		System.out.println("Fifth Constructor Response : " +s5);

		int[] arr4 = new byte[]{65, 66, 67, 68};
		String s6 = new String(arr4, 1, 3);
		System.out.println("Sixth Constructor Response : " +s6);
  }
}

Output →

First Constructor Response :

Second Constructor Response : ABCD

Third Constructor Response : BCD

Fourth Constructor Response : datta

Fifth Constructor Response : att

Sixth Constructor Response : BCD

Restrictions in above Constructors

  • start_index value must be greater than or equals to Zero & less than or equals to length(start_index)-1.
  • length must be greater than or equals to Zero & less than or equals to length(start_index)-start_index.

Note : We will learn about seventh & eight Constructor in StringBuffer and StringBuilder

Example_1 (InstanceMethods.java) →

package com.strings;
import java.util.*;

public class InstanceMethods
{  
	public static void main(String args[])
	{  
		String str = "abcd";
		System.out.println("Character at index 3 is : " + str.charAt(3));
		System.out.println(str + " length is : " + str.length());

		String upper = str.toUpperCase();
		System.out.println("Upper case of " + str + "is :" + upper); // str doesn't changes because String is immutable
		String lower = upper.lowerCase();
		System.out.println("Lower case of " + upper + "is :" + lower); // str doesn't changes because String is immutable
  }
}

Output →

Character at index 3 is : d

abcd length is : 4

Upper case of abcd is ABCD

Lower case of ABCD is abcd

Comparing 'aaa' and 'aaa' value : 0

Comparing 'aab' and 'aaa' value : 1

Comparing 'AAA' and 'aaa' value (ignoring cases) : 0
