MATH CLASS IN JAVA

Math Class

  • Math class methods are used to perform various mathematical operations
  • Math class ia available in java.lang package
  • Math class has only static methods and static data members

    Static Data Members

    • There are only data members in Math class

      → Math.E : 2.718281828459045

      → Math.PI : 3.141592653589793

    Static Methods

    NameDescription
    Math.abs(dataType value)returns absolute number alias positive number of the given dataType value
    → It works for int , long , float , double
    Math.max(dataType v1, dataType v2)returns the maximum of given dataType values
    → It works for int , long , float , double
    Math.min(dataType v1, dataType v2)returns the minimum of given dataType values
    → It works for int , long , float , double
    Math.round(dataType value)returns the nearest long or integer for the given number
    → It works for int , long , float , double
    addExact(dataType v1, dataType v2)returns the sum of its arguments, throwing an exception if the result overflows dataType range
    → It works for int , long
    cbrt(double value)returns the cuberoot of a double value
    copySign(dataType magnitude, dataType sign)returns the first floating-point argument with the sign of the second floating-point argument
    → It works for int , long , float , double
    ceil(double value)returns the largest double value (closest to positive infinity) that is less than or equal to the argument
    decrementExact(dataType value)returns the argument decremented by one, throwing an exception if the result overflows dataType range
    → It works for int , long
    incrementExact(dataType value)returns the argument incremented by one, throwing an exception if the result overflows dataType range
    → It works for int , long
    floor(double value)returns the smallest double value (closest to negative infinity) that is greater than or equal to the argument
    floorDiv(dataType x, dataType y)returns the quotient of both values (either int or long)
    → It works for int , long
    floorMod(dataType x, dataType y)returns the remainder of both values (either int or long)
    → It works for int , long
    multiplyExact(dataType v1, dataType v2)returns the product of its arguments, throwing an exception if the result overflows dataType range
    → It works for int , long
    random()returns a double value with a positive sign, greater than or equal to 0.0 and less than 1.0
    subtractExact(dataType v1, dataType v2)returns the difference of its arguments, throwing an exception if the result overflows dataType range
    → It works for int , long
    pow(dataType v1, dataType v2)returns double value after performing power of these two values
    → It works for int , long , float , double
    sqrt(dataType v1)returns double value which gives square root
    → It works for int , long , float , double
    log(dataType value)returns the natural logarithm which is base e of a double value
    → It works for int , long , float , double
    log10(dataType value)returns the base 10 logarithm of a double value
    → It works for int , long , float , double

    Example_1 (StaticMath.java) →

    package com.maths;
import java.util.*;

public class StaticMath
{  
	public static void main(String args[])
	{  
	  int a = 10, b = 8;
		System.out.println("The maximum value of " + a + " and " + b + " : " + Math.max(a, b));
		
		double c = 7.89, d = 9.69;
		System.out.println("The maximum value of " + c + " and " + d + " : " + Math.max(c, d));

		long e = 10000000000, f = 10000080000;
		System.out.println("The maximum value of " + e + " and " + f + " : " + Math.max(e, f));

		float g = 8.66, h = 11.55;
		System.out.println("The maximum value of " + g + " and " + h + " : " + Math.max(g, h));
  }
}

    Output →

    The maximum value of 10 and 8 : 10

    The maximum value of 7.89 and 9.69 : 9.69

    The maximum value of 10000000000 and 10000080000 : 10000080000

    The maximum value of 8.66 and 11.55 : 11.55
