SCANNER CLASS IN JAVA

Java Scanner Class

  • Scanner Class is available in java.util package.
  • Java provides various ways to read input's from the keyboard, the java.util.Scanner class is one of them.
    • java.util is the package name, Scanner is the class name under this package.
  • The Java Scanner class breaks the input into tokens using a delimiter which is a whitespace by default.
  • Scanner Class provides many methods to read and parse various primitive values.
  • Scanner class extends Object class and implements Iterator and Closeable interfaces.
    public final class Scanner extends Object implements Iterator<String>

Scanner Class Constructors

ConstructorDescription
Scanner(File source)It constructs a new Scanner that produces values scanned from the specified file.
Scanner(InputStream source)It constructs a new Scanner that produces values scanned from the specified input stream.
Scanner(String source)It constructs a new Scanner that produces values scanned from the specified string.
Scanner(Path source)It constructs a new Scanner that produces values scanned from the specified file.

Example_1 (ScannerClass.java) →

package com.scanner;
import java.util.Scanner;

public class ScannerClass
{  
	public static void main(String args[])
	{  
	  Scanner sc = new Scanner(System.in); //InputStream source 
    System.out.print("Enter your name : ");  
    String name = sc.nextLine();  
    System.out.println("Name is: " + name);             
    sc.close();             
  }
}

Input →

Enter your name : Datta Lade

Output →

Name is Datta Lade

Scanner Methods

Method NameMethod Decription
nextInt()Reads input in Integer format
nextDouble()Reads input in Decimal Double format
nextByte()Reads input in Byte format
nextShort()Reads input in Short format
nextLong()Reads input in Long format
next()Reads input in String format without whitespaces
nextLine()Reads input in String format including whitespaces
nextFloat()Reads input in Decimal Float format
nextBoolean()Reads input either true or false
close()It is used to close this scanner.
